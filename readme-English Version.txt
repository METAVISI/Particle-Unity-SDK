Nibiru XR SDK Pro Unity Developer Kit
Version: 2.5.0
Release Date: 2021-01-21

Updates:
1. Add APIs:
a) API to enable/disable Wi-Fi and get its status
b) API to enable/disable Bluetooth and get its status
c) API to manage controllers
d) API to install/uninstall APK without user interface
2. Fix known bugs
3. Add sensor-related APIs


Note:
1. If you are using an old plug-in, it is recommended to remove the old directory first, and then import the new plug-in.

* * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
Please note:

Nibiru XR SDK is designed for Nibiru VR/AR devices. Please use VR/AR devices for development and debugging.

Mobile phone is not supported.

* * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *

Contents Included:

1. Demo: Provide a Unity Demo showing how to use the XR SDK plugin

2. Plugins: XR SDK Unity Plugins that need to be imported into a Unity project for use

3. Doc: Developer guides

4. NibiruXRLauncher: launcher for Nibiru XR applications, which can be used to start XR SDK applications in the mobile phone.
It does not need installing in the HMD.
